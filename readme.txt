付録CD-ROM 操作方法


操作１ 付録CD-ROMをセット

本ディスクをパソコンのCD-ROMドライブにセットして読み込みます。


操作２　indexを開く

次に「index.html」ファイルをダブルクリック、またはご使用のブラウザソフトから開きます。


操作３　トップメニューから見たいアングルを選択

「コスミック・アート・グラフィック　少女ポーズ大全」のトップメニューから各アングルフォト
「切り抜き反転」「回転台反転」「巻末」をクリックするとサムネイル一覧が表示されます。


操作４　各サムネイルをクリックすると拡大されます。

各ボタンは左「prev」(前へ)、上「index」(サムネイル一覧)、右「next」(次へ)に配置されています。


CD-ROMに収録されている画像は全2224カット（サムネイルは含みません）です。
収録画像は「jpeg」形式、解像度72dpi、RGBカラーデータです。
